# Notice Board

A simple PHP notice board application with user login and notice posting features.

## Files

- `admin.php` - Admin dashboard and notice management.
- `config.php` - Database configuration and connection settings.
- `delete_notice.php` - Delete notice functionality.
- `edit_notice.php` - Edit notice functionality.
- `footer.html` - Shared footer markup.
- `header.html` - Shared header markup.
- `index.html` - Landing page.
- `login.php` - User login page.
- `main.php` - Main notice board view.
- `push_notice.php` - Handle notice submission.
- `register.php` - User registration page.

## Setup

1. Place the project in your web server root.
2. Create a MySQL database and update `config.php` with your credentials.
3. Open `register.php` to create an account.
4. Login with `login.php` and manage notices via `admin.php`.
